Cek List Model Gemini

Masukkan API key Gemini kamu lalu klik Get List. Kalau kolom API key dibiarkan kosong, halaman ini akan menampilkan fallback default lokal sebagai referensi.

Siap.

Endpoint yang dipakai adalah GET /v1beta/models. Respons Gemini API bersifat paginated, jadi script ini juga mengambil nextPageToken sampai habis. Model object punya field supportedGenerationMethods, yang bisa dipakai untuk melihat apakah model mendukung generateContent.
